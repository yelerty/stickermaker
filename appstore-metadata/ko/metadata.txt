스티커 메이커를 심사해 주셔서 감사합니다!

이 앱은 배경 제거를 위해 기기 내 AI (Vision Framework)를 사용합니다. 데이터 수집이나 전송이 없습니다. 모든 처리는 로컬에서 이루어집니다.

테스트 계정: 불필요 (로그인/계정 시스템 없음)

기능 테스트 방법:
1. 요청 시 사진 라이브러리 권한 허용
2. "스티커" 탭 시도 - 사진 선택, 배경 자동 제거
3. "사진 GIF" 탭 시도 - 2-10장의 사진 선택하여 GIF 생성
4. "비디오 GIF" 탭 시도 - 비디오 선택, 구간 선택, GIF 생성
5. "배경 합성" 탭 시도 - 사람 사진 + 배경 사진 선택

앱은 100% 오프라인으로 작동합니다. 개인정보 처리방침, 이용약관, 지원 페이지는 저장소에 문서화된 대로 GitHub Pages에 호스팅되어 있습니다.
